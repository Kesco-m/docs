Возвращает экземпляр интерфейса T:System.Web.IHttpHandler для обработки требуемого ресурса.

Возвращает
Новый обработчик T:System.Web.IHttpHandler, обрабатывающий запрос; в противном случае — null.
Аргументы
contextЭкземпляр класса T:System.Web.HttpContext, который предоставляет ссылки на внутренние объекты сервера (например, Запрос, Ответ. Сеанс и Сервер), используемые для обслуживания HTTP-запросов.
requestTypeМетод http для передачи данных (GET или POST), используемый клиентом.
virtualPathВиртуальный путь к требуемому ресурсу.
pathСвойство P:System.Web.HttpRequest.PhysicalApplicationPath требуемого ресурса.

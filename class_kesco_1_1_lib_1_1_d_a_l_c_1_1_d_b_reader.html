Подробное описание

Класс абстрагирует работу с несколькими классами SqlDataReader и SqlConnection, объединяя их в единое целое.

Паттерн проектирования "фасад". Рекомендуется использовать в конструкции "using" Существует программа, преобразования запроса или ХП в класс сущности и начитки данных: https://titan.kescom.com:8443/svn/web/GeneratorORM

Примеры использования и юнит тесты: Kesco.App.UnitTests.DalcTests.DalcTest

Конструктор DBReader

Для получения OUTPUT параметров необходимо принудительно закрыть ридер - вызвать метод Close(), до закрытия параметры остаются без значения, это происходит по тому что Return и OUTPUT параметры формируются по окончанию работы ХП, пока открыт SqlDataReader работа ХП считается не завершенной.

Аргументы
comdTextТекст запроса или название ХП
typeТип команыды - текст или хранимая процедура
connectionStringСтрока подключения
parametersВходные значения
parametersOutВыходные параметры, для хп автоматически формируется возвращаемое значение()
